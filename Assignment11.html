CSS Positioning Techniques Demo

1. Static Positioning (Default)

This is the default positioning. Elements are positioned according to the normal flow of the document.

Static

2. Relative Positioning

Relative positioning moves an element relative to its normal position.

Relative

3. Absolute Positioning

Absolute positioning positions an element relative to its nearest positioned ancestor (or the initial containing block if there is no positioned ancestor).

Absolute

4. Fixed Positioning

Fixed positioning positions an element relative to the browser window. It will stay in the same place even when the page is scrolled.

Fixed

5. Sticky Positioning

Sticky positioning allows an element to "stick" when it reaches a specified threshold position while scrolling.

Sticky
